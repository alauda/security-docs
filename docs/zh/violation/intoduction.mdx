# 介绍

## 什么是政策违规？

Alauda Container Security for Kubernetes 允许您查看、调查和处理集群中的政策违规。您可以快速识别违规的根本原因，并采取纠正措施以改善您的安全态势。

## 如何检测违规

Alauda Container Security 的内置政策可以检测广泛的安全问题，包括：

- 漏洞（CVE）
- DevOps 最佳实践的违规
- 高风险的构建和部署活动
- 可疑的运行时行为

您可以使用默认的安全政策或定义自己的自定义政策。当启用的政策被违反时，Alauda Container Security 会将其报告为违规，供您审查和修复。
