# 介绍

Alauda Kubernetes 容器安全平台旨在帮助您识别、评估和管理容器化环境中的安全风险。它提供了对漏洞、错误配置和风险运行活动的可见性，使您能够优先处理和解决部署中最关键的安全问题。

## 风险视图

**风险**视图显示了所有集群中的所有部署，按综合风险指标排序。该指标考虑了政策违规、镜像内容、部署配置和其他相关因素。风险最高的部署出现在列表的顶部。

### 部署属性

**风险**视图中的每个部署包括以下属性：

| 属性         | 描述                                             |
| ------------ | ------------------------------------------------ |
| **名称**     | 部署的名称                                      |
| **创建时间** | 部署的创建时间                                  |
| **集群**     | 部署运行所在的集群名称                          |
| **命名空间** | 部署所在的命名空间                              |
| **优先级**   | 基于严重性和风险指标的优先级排名              |

### 风险视图中的操作

您可以在**风险**视图中执行以下操作：

- 按任意列升序或降序排序部署。
- 使用过滤栏过滤部署。
- 根据过滤条件创建新策略。
- 选择一个部署以查看详细的风险信息。

通过分析**风险**视图，您可以快速识别和解决环境中最关键的安全风险。

## 如何访问风险视图

- 前往 Alauda 容器安全门户，从导航菜单中选择**风险**。
