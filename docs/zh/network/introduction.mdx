# 介绍

[Kubernetes 网络策略](https://kubernetes.io/docs/concepts/services-networking/network-policies/) 是一种规范，定义了一组 Pod 之间以及与其他网络端点之间的通信方式。这些网络策略以 YAML 文件的形式进行配置。仅通过查看这些文件，通常很难判断所应用的网络策略是否达到了预期的网络拓扑。

Alauda 容器安全性收集了来自您的调度器的所有定义的网络策略，并提供工具以简化这些策略的使用。

为了支持网络策略的执行，Alauda 容器安全性提供了以下工具：

- 网络图
- 网络策略生成器
- 网络策略模拟器
- 构建时网络策略生成器
