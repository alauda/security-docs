# 与通用 Docker 注册表集成

本指南解释了如何将 Alauda 容器安全性与 Docker 注册表或 Harbor 集成。

## 先决条件

- 获取用于与 Docker 注册表或 Harbor 进行身份验证的用户名和密码。

## 集成步骤

1. 在 Alauda 容器安全性门户中，导航到 **平台配置** > **集成**。
2. 在 **镜像集成** 部分，选择 **通用 Docker 注册表**。
3. 点击 **新建集成**。
4. 填写以下字段：
   - **集成名称**：输入此集成的名称。
   - **端点**：指定注册表地址。
   - **用户名** 和 **密码**：输入您的凭证。
5. （可选）如果您不使用 TLS 证书连接到注册表，请选择 **禁用 TLS 证书验证（不安全）**。
6. 要验证连接，请点击 **测试**。如果您希望跳过测试，请选择 **不测试直接创建集成**。
7. 点击 **保存** 完成集成。
