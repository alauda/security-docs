# 介绍

Alauda 容器安全提供自动化合规检查，帮助您的 Kubernetes 集群满足行业标准和监管要求。通过持续扫描您的环境与 CIS、PCI-DSS、HIPAA 等基准进行对比，它使您能够高效识别和解决合规性差距。

Alauda 容器安全支持多种合规框架，包括 CIS、HIPAA、NIST、PCI DSS 和 Alauda 容器平台合规操作员配置文件，允许您：

- 评估并展示您基础设施的监管合规性。
- 加强您 Kubernetes 环境的安全态势。
- 清晰了解集群、命名空间和节点的合规状态。

通过利用这些能力，您可以主动管理合规风险，简化审计流程，并确保您的容器化工作负载按照最佳实践和监管要求运行。
